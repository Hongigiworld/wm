# Weekly Meal - Homekeeper

[🍽️ 앱 열기](https://hongigiworld.github.io/recipe/)

## 개요

한국 가정식 기반 주간 식단 플래너. 냉장고 재료, 식단 스타일, 영양 정보를 기반으로 식단을 자동 생성합니다.

## 구조

단일 파일 구조 (`index.html`)

- CSS, JS, 데이터(MENU_DB, NUTRITION_DB 등) 전부 포함
- localStorage 기반 데이터 저장
- 외부 의존성: Pretendard 폰트 (CDN), Twemoji (CDN)

## 주요 기능

- A/B/C Flow 기반 식단 생성
- 냉장고 재료 관리 및 유통기한 추적
- 장보기 목록 자동 생성
- 영양 정보 (칼로리, 단백질, 탄수화물, 지방) 표시
- 식단 다이어리
